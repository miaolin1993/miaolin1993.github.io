东派紫微--禄逢冲破

作者：林学风

详细解释下在东派紫微斗数体系里，禄逢冲破是如何看的，关于怎么解，可以去内部群里自行查看内部资料

禄逢冲破，如其文字意思，好好的禄，被破坏掉了，禄可以理解成家里有一个金库，冲破就是有人拿炸药包把金库给炸了

这段话里有三个角色关系：

  • 禄是谁？
  • 炸药是谁？
  • 投炸药的人是谁？

理解了这三个角色关系，也就能很清楚的明白了，

禄，是指生年禄或者禄存，流年的大限的那些统统不算，讨论冲破时，就只看这两个

炸药，指生年忌、向心/离心自化忌和宫干飞化的忌

投炸药的人，就下面三种情况：

  • 同宫
  • 对宫
  • 飞宫

重点：谁投炸药，就是谁冲破，一个盘是会有很多个宫位禄逢冲破的

本宫引发

指生年禄或禄存，和生年忌或向心/离心忌同宫，这种属于自己禄逢冲破

生年忌同宫

图一：禄存和生年忌同宫

这属于迁移宫禄逢冲破

图二：生年禄和生年忌同宫

这属于福德宫禄逢冲破

离心忌同宫

禄：是指生年禄或者禄存，两个哪个都行，后面就不赘述了

这也属于迁移宫禄逢冲破

向心忌同宫

疾厄宫引发的向心忌，去了对面还会冲回来，这属于疾厄宫的禄逢冲破，还是属于本宫引发的范畴

对宫引发

指禄的宫位对面，存在生年忌，向心忌和离心忌的情况，这种属于对宫的禄逢冲破

跟本宫不同的是，谁来引动的炸药，就是谁的冲破，这句话很重要，看下面图

生年忌在对宫

禄在夫妻宫，对宫官禄宫有生年忌，因为是官禄宫的生年忌冲了禄，所以叫官禄宫的禄逢冲破

关键点是要找到谁是引爆炸药的人，自己宫位就是自己，对宫就是对宫，飞过来的就看是从谁那飞来的，就是谁

向心忌在对宫

跟上一个类似，禄在子女宫，而田宅宫有向心忌，冲了禄，所以就是田宅宫的禄逢冲破

离心忌在对宫

禄在交友，兄弟有离心忌，会冲交友宫的禄，由兄弟宫引发，所以叫兄弟宫的禄逢冲破

飞宫引发

宫干引发直冲

田宅宫，丙干引发廉贞化忌去了生年禄的位置，这时候田宅宫就是引爆炸药的人，所以叫田宅宫的禄逢冲破

宫干引发对宫冲

夫妻宫引发迁移文昌化忌冲了命宫的禄，这情况叫夫妻宫的禄逢冲破

最后再强调下，禄：是指生年禄或者禄存。这两个谁都可以，流年大限那些都不算

总结

理解了上面说的三者关系，找禄逢冲破就很简单

  • 第一步：找到禄存和生年禄的位置
  • 第二步：就十二个宫，挨个看看是谁冲了这俩就行了

关于具体冲破啥意思，怎么解？内部群里找资料或是群里问，就不在公开场合写了
