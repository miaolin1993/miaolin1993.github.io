关于命理

一命二运三风水，四积阴德五读书，六名七相八敬神，九交贵人十养生。

每个人都有自己的属性：

  • 有的人业务能力强，口才好，擅长结交人脉
  • 有的人钻研能力强，肯研究，解决各种疑难杂症

类似于打游戏，有坦克，有治疗，有输出一样，各司其职，都有属于自己的位置

  • 在自己擅长的领域，才能很好的发挥
  • 想要很好的发挥，就要找到自己擅长的领域

学习命理的目的只有两个：

  1. 更好认识自己
  2. 认识规律
