## VitePress 的优势

VitePress 是一个静态站点生成器，具有以下优势：

1. **极速构建** - 基于 Vite，构建速度非常快
2. **Vue 3 驱动** - 使用 Vue 3 和 Vite 构建
3. **Markdown 支持** - 支持 Markdown 和 Vue 组件
4. **主题定制** - 易于定制和扩展
